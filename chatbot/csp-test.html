CSP Testing Instructions:

  1. Make sure your chatbot server is running on http://localhost:5173
  2. Open this test file in a browser
  3. The iframe should load successfully
  4. Check browser console for any CSP violations
  5. To test blocking, try embedding from a domain not in the allowlist

How to Add/Remove Allowed Domains:

  1. Edit the CSP meta tag in your index.html
  2. Modify the frame-ancestors directive
  3. Add/remove domains as needed
  4. Rebuild and restart your server
