💰 PAYMENT FLOW
═══════════════════════════════════════════════════════════════════════════════

1. USER COMPLETES COURSE
   └─ Course status: "completed"

2. ADMIN/TRAINER GENERATES CERTIFICATE
   └─ POST /api/v1/certificates/generate
   └─ Certificate created with isPaid: false

3. USER VIEWS CERTIFICATE
   └─ GET /api/v1/certificates/my-certificates
   └─ Sees "Payment Required - Rs. 50/-"

4. USER CLICKS "PAY NOW"
   └─ POST /api/v1/payments/create-order
   └─ Backend creates Razorpay order
   └─ Returns order_id, amount, key

5. RAZORPAY CHECKOUT OPENS
   └─ User enters payment details
   └─ Completes payment

6. RAZORPAY RETURNS RESPONSE
   └─ razorpay_order_id
   └─ razorpay_payment_id
   └─ razorpay_signature

7. FRONTEND VERIFIES PAYMENT
   └─ POST /api/v1/payments/verify
   └─ Backend verifies signature (CRITICAL)
   └─ Updates database if valid

8. DATABASE UPDATED
   └─ Payment.status = "completed"
   └─ Certificate.isPaid = true

9. USER DOWNLOADS CERTIFICATE
   └─ GET /api/v1/certificates/:id/download
   └─ Backend checks isPaid === true
   └─ Returns certificate PDF URL
