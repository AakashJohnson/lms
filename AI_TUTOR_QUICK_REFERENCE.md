## 💬 How to Ask Questions

### ✅ Good Questions:
```
✓ "Explain React hooks in detail with examples"
✓ "Show me step-by-step how to create a REST API"
✓ "What's the difference between SQL and NoSQL?"
✓ "Give me real-world examples of design patterns"
```

### ❌ Avoid:
```
✗ "Tell me about programming"  (too broad)
✗ "Help"  (not specific)
✗ "What is this?"  (no context)
```

## 📚 Question Types

### 1. **Deep Understanding**
**Keywords**: explain, why, how does, what is, difference
```
"Why does JavaScript use prototypal inheritance?"
"How does async/await work internally?"
```
**You Get**: 800 tokens, comprehensive explanation

### 2. **Examples**
**Keywords**: example, demonstrate, show me, illustrate
```
"Show me examples of React hooks"
"Demonstrate error handling in Node.js"
```
**You Get**: 700 tokens, multiple examples

### 3. **Step-by-Step**
**Keywords**: step, process, how to, guide
```
"How to deploy a React app step by step"
"Guide me through database design"
```
**You Get**: 800 tokens, detailed steps

## 🎓 Pro Tips

### 1. **Use Course Context**
```
1. Select course from dropdown
2. Ask course-specific questions
→ Get accurate, curriculum-based answers
```

### 2. **Upload Documents**
```
1. Click attachment icon
2. Upload PDF/DOC/TXT
3. Ask questions about it
→ AI analyzes and answers from document
```

### 3. **Follow-up Questions**
```
First: "What are closures?"
Then: "Give me an example"
Then: "How is this used in React?"
→ AI remembers context
```

### 4. **Specify Format**
```
"Explain like I'm 5"
"Give technical details"
"Show with code examples"
"Use analogies"
```

## 🐛 Troubleshooting

### "AI is currently busy"
→ Try shorter question or wait few seconds

### Generic answers
→ Select course context or add more details

### Slow response
→ Normal for complex questions (uses better model)

### Error message
→ Check internet, retry, or rephrase
